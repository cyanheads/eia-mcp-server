# 0.2.1 — 2026-05-25

## Fixed

- **`eia_query_route`** no longer returns empty data when called without a prior `eia_describe_route`. The service now fetches and caches route metadata on-demand when the cache is cold, so cold-start queries return full results. ([#17](https://github.com/cyanheads/eia-energy-mcp-server/issues/17))
