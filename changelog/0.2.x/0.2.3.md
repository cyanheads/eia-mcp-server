# 0.2.3 — 2026-05-30

## Changed

- **`eia_search_routes`** — query echo (`effectiveQuery`), `totalIndexed`, and an optional `notice` for empty results now surface in a typed `enrichment` block reaching both `structuredContent` JSON and the `content[]` markdown trailer. `total_indexed` removed from `output` (value preserved in `enrichment.totalIndexed`).
- **`eia_query_route`** — `effectiveRoute`, `totalCount`, `returnedCount`, and `appliedFilters` now in `enrichment`; `appliedFilters` renders as structured markdown via `enrichmentTrailer`. `total` and `returned_count` removed from `output` (values preserved in `enrichment.totalCount` / `enrichment.returnedCount`).
- **`eia_dataframe_query`** — `totalRows`, `returnedRows`, and a capped-result `notice` now in `enrichment`. `row_count` removed from `output` (value preserved in `enrichment.totalRows`). `rows[]` schema tightened to `z.object({}).passthrough()` with per-row description.
- **`@cyanheads/mcp-ts-core`** ^0.9.13 → ^0.9.16.
